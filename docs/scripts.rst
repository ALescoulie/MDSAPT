MD-SAPT Scripts
===============

MD-SAPT is packaged with some scripts to aid users.

`mdsapt_get_runinput`
_____________________

Usage

.. code-block:: bash
    python mdsapt_get_runinput.py filename

`mdsapt_run_sapt`
_________________

Usage

.. code-block:: bash
    python mdsapt_run_sapt.py input.yaml results.out
